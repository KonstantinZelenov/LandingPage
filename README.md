**Персональный сайт-портфолио тренера по фехтованию, каскадёра и спортсмена**

Интерактивный веб-сайт с вращающимся колесом меню, поддержкой двух языков (RU/EN), переключением темной/светлой темы и формой обратной связи. Проект демонстрирует услуги постановки боевых сцен для кино и театра, а также тренировок по артистическому фехтованию.

## ✨ Особенности

| Функция | Описание |
|---------|----------|
| 🎨 **Темная/светлая тема** | Переключение темы с сохранением выбора в `localStorage` |
| 🌍 **Двуязычность** | Полный перевод интерфейса на русский и английский |
| 🗡️ **Анимированное меню** | Вращающееся колесо с мечами и плавной навигацией |
| 📱 **Адаптивный дизайн** | Корректное отображение на всех устройствах с использованием `clamp()` и медиа-запросов |
| 📧 **Форма обратной связи** | Валидация полей и отправка данных через API |
| 💰 **Калькулятор прайс-листа** | Выбор типа тренировки с автоматическим заполнением данных |
| ⚡ **Оптимизация** | Адаптивные изображения (WebP/JPEG), lazy-loading, безопасные CSS-переменные |

## 🚀 Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- npm или yarn

### Локальный запуск

1. **Клонируй репозиторий**
```bash
git clone https://github.com/yourusername/konstantin-zelenov.git
cd konstantin-zelenov
Установи зависимости

bash
npm install
Настрой переменные окружения
Создай файл .env в корне проекта:

env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
Запусти сервер

bash
# Для разработки
npm run dev

# Для production
npm start
Открой в браузере

text
http://localhost:3000
Альтернативный запуск (только фронтенд)
Если бэкенд не требуется, просто открой index.html в браузере:

bash
# Используй Live Server в VS Code
# или любой другой статический сервер
npx serve .
🔌 API Эндпоинты
POST /api/send-form
Отправка данных формы обратной связи.

Request body:

json
{
  "name": "Иван Иванов",
  "mail": "ivan@example.com",
  "lesson_type": "SW",
  "message": "Хочу записаться на тренировку",
  "trainingType": "personal",
  "duration": "1 hour",
  "price": "50 usd",
  "description": "Hall rent is paid separately..."
}
Response:

json
{
  "success": true,
  "message": "Email sent successfully"
}
Ошибки:

400 — Невалидные данные

500 — Ошибка сервера
